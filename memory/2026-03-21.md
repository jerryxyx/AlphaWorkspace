## 🛠️ Technical Setup & Learnings

### x-reader Installation & Authentication
- **x-reader version**: runesleo/x-reader (Python CLI, not AbMoeiz Windows app)
- **Installation method**: `pipx install x-reader` (global, isolated environment)
- **Browser dependencies**: Playwright + Chromium via `pipx inject x-reader "x-reader[browser]"`
- **Authentication workflow**: `x-reader login xhs` opens browser for Xiaohongshu login, saves session to `~/.x-reader/sessions/xhs.json`
- **Session persistence**: Once logged in, future requests use saved session headlessly (no browser window)

### PATH Issues with Conda Environments
- **Problem**: `x-reader` command not found in conda `(base)` environment
- **Root cause**: Conda prepends its own bin directories, pushing `~/.local/bin` down in PATH
- **Solutions**: 
  - Use full path: `/Users/xyx/.local/bin/x-reader`
  - Add `export PATH="$HOME/.local/bin:$PATH"` to `~/.zshrc` before conda initialization
  - Use `pipx run x-reader` (works in any environment)

### Xiaohongshu Access Patterns
- **Jina fetch fails**: 451 "Unavailable For Legal Reasons" error for Xiaohongshu URLs
- **Browser fallback works**: Playwright with saved session successfully fetches pages
- **Search vs Content**: x-reader fetches pages as content; search result pages have minimal text (mostly images/links)
- **Need specific post URLs**: For actual travel notes, need URLs like `https://www.xiaohongshu.com/explore/abc123`
