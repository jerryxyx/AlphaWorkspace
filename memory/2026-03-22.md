```
1. User provides consumer key
2. Search tools: COMPOSIO_SEARCH_TOOLS (session_id: "safe")
3. Manage connections: COMPOSIO_MANAGE_CONNECTIONS (toolkits: ["notion"])
4. Wait for auth: COMPOSIO_WAIT_FOR_CONNECTIONS
5. Execute tools: COMPOSIO_MULTI_EXECUTE_TOOL
```

### Notion API Locking Behavior

- **Implication**: Page locking is a **UI‑only curation step** that users must perform manually in the Notion client.
- **Best practice**: When creating pages that should be locked (e.g., final itineraries, reference documents), instruct the user to lock them manually after review, or rely on the API's default unlocked state for collaborative editing.
