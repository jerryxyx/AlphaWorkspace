# 2026‑03‑25 · Early‑Exercise Boundary Pattern Analysis & Premium Plots

## 🔍 Boundary Pattern Analysis (σ_spot = σ_fair)
**Objective**: Determine if call/put spreads exhibit one or two early‑exercise boundaries (finite range around strike) under varying volatility parameters.

**Parameter grid**:
- σ_fair ∈ [0.20,0.40] step 0.01 (21 values)
- σ_offset ∈ [0.00,0.10] step 0.01 (11 values)
- σ_spot = σ_fair (spot volatility equals implied volatility)
- S₀ = K = 100.0, T = 1.0, r = 0.05
- Lattice steps n = 50

**Method**: Fast inspection of exercise‑flag patterns from binomial tree at mid‑life (τ=0.5) and near expiry (τ=0.98).

**Findings**:
1. **Mid‑life (τ=0.5)**: **All 462 combos** (21×11×2) show pattern **“always”** – immediate unwinding optimal for every spot level. No finite boundary.
2. **Near expiry (τ=0.98)**:
   - 431 combos remain “always”.
   - **31 combos show “two_regions”** – exercise optimal at both high and low spots, with a small non‑exercise gap in between (e.g., spots >136.5 and spots <121.9).  
     → This is **not** a single contiguous range around the strike.
3. **Early‑exercise premium**: **Zero** for all combos – because S₀ = K lies inside the exercise region. American spread value equals European spread value.

**Conclusion**: Under equal spot and implied volatility, there is **no finite exercise boundary around the strike**; immediate unwinding is optimal for all spots. A finite contiguous range emerges only when σ_spot > σ_fair (higher spot volatility).

## 📊 Cross‑Method Validation (Lattice, PDE, Monte Carlo)

**Objective**: Cross‑validate three numerical methods for pricing American‑style call/put spreads under volatility mispricing.

**Parameter grid**:
- σ_fair = 0.2 (constant)
- σ_spot ∈ {0.1, 0.5} (below and above σ_fair)
- σ_offset ∈ {0.0, 0.05}
- Spot grid: 11 points from 50 to 150
- S₀ = K = 100.0, T = 1.0, r = 0.05
- Numerical settings: lattice n=50, MC paths=300 (LSM), PDE grid 80×30

**Key findings**:
1. **Method agreement**:
   - **Lattice and Monte Carlo match exactly** (mean difference = 0.000000).
   - **PDE shows small systematic undervaluation** (~0.0016) due to grid discretization and zero‑boundary‑condition treatment.
2. **Finite exercise interval** appears **only when σ_spot > σ_fair**:
   - σ_spot=0.5, σ_offset=0.05 → exercise region **[73.0, 124.3]** around K=100.
   - Outside this band, waiting is valuable (negative premium).
3. **Early‑exercise premium = 0 when S₀ = K** – immediate unwinding optimal at‑the‑money.
