## 📈 Lattice Enhancement
- Added `black_scholes_put` function to `src/lattice.py`.
- Modified `binomial_tree_value` to accept `option_type` parameter (`'call'` or `'put'`).
- Boundary definition:
  - **Call spreads**: boundary = smallest spot where unwind ≥ continuation (exercise region = spots **above** this level).
  - **Put spreads**: boundary = largest spot where unwind ≥ continuation (exercise region = spots **below** this level).

## 🔍 Premium Analysis (Extreme OTM Case)
**Parameters**: `K=200.0`, `S0=100.0`, `σ_fair=0.8`, `σ_offset=0.3`, `σ_spot=0.9`, `T=1.0`, `r=0.10`

**Findings**:
1. **Premium positive everywhere** – unwind value > continuation value for all spots (50–300) at all times.
   - Immediate unwinding is optimal across the entire spot range.
   - No finite breakeven point within the range.
2. **Monotonicity**:
   - Call‑spread premium increases with spot (positive delta).
   - Put‑spread premium also increases with spot (vega dominates when spot is far below strike).
3. **Boundary scaling** – earlier tests confirm that normalised boundary B/S₀ depends on moneyness (S₀/K), not on S₀ alone.

## ✅ Verification
- Lattice boundary returns the spot at the *lowest node* where exercise is optimal for calls, and at the *highest node* for puts.
- American values for call and put spreads are identical (within numerical error) under constant volatility, as expected from put‑call parity.
