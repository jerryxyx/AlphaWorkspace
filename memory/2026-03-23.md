# 2026‑03‑23 · Warrant‑Vol‑Management Numerical Suite

## Project Delivery
Built a three‑method numerical suite for pricing the arbitrage derivative (long high‑vol European call, short low‑vol European call, early unwinding allowed).

**Methods implemented:**
1. **Lattice (binomial tree)** – CRR tree with early‑exercise boundary output.
2. **Monte Carlo LSM** – Antithetic variates, regression basis `[1, S, S²]`.
3. **PDE (finite difference)** – Implicit Euler with PSOR for early‑exercise constraint (boundary detection needs tuning).

**Volatility‑function library** (`config.py`): constant, smile, skew, step.

**Jupyter notebook** (`notebook/test.ipynb`) with binomial‑tree visualization (green = unwind, red = wait).

## Key Learnings
- **Exercise boundary** ≠ **early‑exercise premium**.
  - Boundary = spot level above which unwind ≥ continuation (for calls).
  - Premium = extra value from optionality to wait, measured at initial spot S₀.
- With extreme parameters (`σ_fair=0.8`, `σ_offset=0.3`, `K=200`, `S₀=100`, `r=10%`):
  - Boundary at t=0 ≈ 99.9.
  - S₀=100 is above boundary → immediate unwinding optimal → premium = 0.
- **Call spreads** increase with spot (positive delta) → exercise region above boundary.
  *Put spreads would reverse inequality.*
- Boundary depends on **strike K**, not initial spot S₀. S₀ only determines where we start on the boundary curve.
- When options are OTM (S₀ far from K), vega small, time decay dominates → boundary low.
  When options are ATM, vega large → boundary expected to be closer to K.

## Boundary at Maturity & One Step Before
- **At maturity (time to expiry = 0)**: Both legs have identical payoff max(S‑K,0) → spread payoff = 0 → unwind value = 0.
- **One step before expiry**: Continuation value = discounted expectation of maturity values = 0.
- Unwind value > 0 for all spots (high‑vol call more expensive than low‑vol call) → unwind > continuation → **exercise optimal at all spots**.
- Therefore, boundary at one step before expiry is the smallest possible spot (≈0). Lattice shows exercise flags True for all nodes at step n‑1.
